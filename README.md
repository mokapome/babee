## 프로젝트 소개

스프링부트+JSP를 사용하여 개발한 육아용품 쇼핑몰 프로젝트입니다. 이 프로젝트는 간단하고 직관적인 UI로 사용자에게 편의성을 제공하며, 다양한 커뮤니티와 육아에 도움이 되는 정보를 제공합니다.
